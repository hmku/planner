Methodology

Historical Data

Fixed-beta simulations sample contiguous 5-year historical blocks from 1950-2025. Dynamic-beta simulations sample one historical year at a time so each beta choice uses only information available before that year's return is drawn.

Return Model

Portfolio nominal return is the sampled T-bill return plus SPY beta times the sampled S&P 500 excess return over T-bills, then converted into current dollars by dividing by that year's inflation factor. Dynamic beta chooses from 0.0 to 1.5 in 0.1 steps.

Dynamic Beta

Dynamic mode builds a backward dynamic-programming policy over plan year and current wealth. It uses a zero bucket plus 120 log-spaced positive wealth buckets from $10,000 to $1 billion, picks the beta with the lowest estimated depletion probability, then breaks ties by highest expected terminal wealth.

Cash Flows

Income and expenditure amounts are treated as annual current-dollar flows. Start and end years can be fixed or tied to the plan's current year and expected year of death.

Timing

Within each simulated year, returns compound continuously and net cash flow is spread continuously through the year. Depletion occurs when current-dollar wealth reaches zero or below.

Charts

The depletion-year chart shows probability across all simulations. The overview charts show expected current-dollar wealth, average SPY beta, and downsampled paths for hover inspection. Dynamic-beta runs expose the policy's wealth buckets and recommended beta by year in Inspect Beta Policy.

Limits

The model is a historical bootstrap, not a forecast. It does not model taxes, account types, asset allocation beyond SPY beta, fees, changing spending behavior, sequence constraints, or valuation-based expected returns.
