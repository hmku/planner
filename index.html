Methodology

Historical Data

The simulator samples contiguous 5-year historical blocks from 1928-2025. Within each block, annual S&P 500 total returns, 3-month T-bill returns, and CPI inflation stay paired and in historical order.

Return Model

Each simulation path is built by sampling 5-year historical blocks with replacement, truncating the final block if needed. Portfolio nominal return is the sampled T-bill return plus SPY beta times the sampled S&P 500 excess return over T-bills, then converted into current dollars by dividing by that year's inflation factor.

Cash Flows

Income and expenditure amounts are treated as annual current-dollar flows. Start and end years can be fixed or tied to the plan's current year and expected year of death.

Timing

Within each simulated year, returns compound continuously and net cash flow is spread continuously through the year. Depletion occurs when current-dollar wealth reaches zero or below.

Charts

The depletion-year chart shows probability across all simulations. The net-worth chart shows expected current-dollar wealth across all paths plus a downsampled set of individual paths for hover inspection.

Limits

The model is a historical bootstrap, not a forecast. It does not model taxes, account types, asset allocation beyond SPY beta, fees, changing spending behavior, sequence constraints, or valuation-based expected returns.
